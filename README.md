# Autenticação API

API de autenticação construída com **Node.js**, **Express**, **MongoDB** e **JWT**, rodando em containers Docker.

## Tecnologias
- Node.js + Express
- MongoDB + Mongoose
- JWT para autenticação
- Docker + Docker Compose
- Jest para testes automatizados
